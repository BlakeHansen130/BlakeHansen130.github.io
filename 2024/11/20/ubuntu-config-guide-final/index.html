2.1 截图界面快捷键

  • Space: 切换左侧工具栏
  • Ctrl + C: 复制截图到剪贴板

  • Mouse Wheel: 更改工具笔画宽度
  • Shift: 在绘制时保持正方形/圆形
  • Esc / Delete: 退出截图
  • Return: 确认截图

1. Conky 常见问题

1.1 网速显示为 0

  • 现象：网络速度一直显示为 0，即使有实际流量
  • 原因：
    • 网络接口名称配置错误
    • 动态接口检测命令解析问题
  • 解决方案：
    1. 检查实际网络接口名称，并确认活动接口（见上文网络配置部分）
    2. 更新配置文件中的接口名称：
